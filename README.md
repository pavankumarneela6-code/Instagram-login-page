# 📸 Instagram Login Page Clone

## 📖 Project Description

This project is a simple Instagram Login Page Clone developed using HTML and CSS. It recreates the Instagram login interface with username/email input fields, password field, Facebook login option, forgot password link, sign-up section, and footer.

## 🚀 Features

* Instagram Logo Section
* Username / Email Input Field
* Password Input Field
* Login Button
* OR Divider
* Login with Facebook Option
* Forgot Password Link
* Sign Up Link
* Responsive Layout
* Background Image
* Footer Section

## 🛠️ Technologies Used

* HTML5
* CSS3
